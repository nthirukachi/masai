# MQ9 Learning Rate Comparison on Fashion-MNIST

## Project Overview
This project compares the effect of two different learning rates (1e-3 and 5e-4) on the training of a Multi-Layer Perceptron (MLP) using the Fashion-MNIST dataset.

## Structure
- `src/`: Python source code
- `notebook/`: Teaching-oriented Jupyter Notebook
- `documentation/`: Detailed explanations, observations, and exam prep
- `slides/`: Presentation slides
- `outputs/`: Execution logs and plots

## How to Run
```powershell
cd c:\masai\MQ9_Learning_Rate_Comparison_FashionMNIST
uv run python src/learning_rate_comparison.py
```
