# MQ7 Silhouette Score Analysis

## Project Overview
This project evaluates the Silhouette Score on real retail data (Mall Customer Segmentation) to determine the optimal number of clusters for customer segmentation.

## Structure
- **notebook/**: Jupyter Notebook for teaching.
- **src/**: Python source code.
- **documentation/**: Detailed explanations, interview/exam prep.
- **slides/**: Presentation slides.
- **outputs/**: Execution results and plots.

## Usage
Run the python script:
```powershell
uv run python src/silhouette_analysis.py
```
